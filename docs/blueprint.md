# **App Name**: Breakpoint

## Core Features:

- Problem Set Access (Practice): Learners can access and browse a library of debugging scenarios.
- Problem Solving Environment: Highlight logs, submit diagnoses and next steps.
- Scoring and Feedback: Receive scores on submissions and unlock detailed explanations to understand correct reasoning.
- Assessment Assembly (Company): Companies can select problems, set time limits and invite candidates to assessments.
- Interaction Tracking: Capture a rich interaction trace during assessment attempts.
- AI Debugging Analysis Tool: Analyze interaction telemetry using AI to infer reasoning quality, focus, and blind spots.
- Recruiter-Ready Report Generation: Generate a report summarizing root-cause accuracy, debugging judgment, and behavioral patterns of a candidate based on AI analysis of interaction data, suitable for recruiters.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and focused feel.
- Background color: Light Gray (#F5F5F5), nearly white, for clean readability.
- Accent color: Teal (#009688) to highlight key actions and information.
- Body and headline font: 'Inter' sans-serif for a modern, objective look; highly readable.
- Use crisp, technical icons related to debugging concepts and system components.
- Implement a clean, segmented layout for problem descriptions, log sections, and diagnostic prompts.
- Use subtle transitions and loading animations to maintain a smooth, professional user experience.
